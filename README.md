# Annotation-and-ChatBot-Training

*Annotation-and-ChatBot-Training* is a toolkit for building chatbots: it allows you to annotate NLU data (entities, tokens), store annotations, and run a training pipeline — useful for building NLP / conversational-AI systems.

## Features

- ✅ Entity / intent annotation tool with user-friendly interface  
- ✅ Tokenization and NLU data preparation  
- ✅ Storage of annotations (entities, intents) in structured format  
- ✅ Chatbot / NLU training pipeline (using Node.js / Python / relevant libraries)  
- ✅ Test suite included to verify expected behavior  
- ✅ Clear directory structure to separate UI, data, server, tests  

## Tech Stack

- Python  
- JavaScript (Node.js)  
- HTML / CSS for UI  
- JSON / YAML for configuration
